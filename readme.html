Projet WordPress - TREASURES

Ce dépôt contient le code source de mon projet WordPress. Il s'agit d'un site conçu pour le commerce électronique, spécialisé dans la vente de produits de mode et d'accessoires, notamment des sacs et des bijoux.

Fonctionnalités principales

  • Créé avec WordPress pour une gestion facile du contenu.
  • Utilisation du thème Blocksy pour un design moderne et flexible.
  • Plugins intégrés pour améliorer les performances et les fonctionnalités du site :
    • Elementor : Construction de pages avec un éditeur visuel intuitif.
    • Premium Addons for Elementor : Widgets supplémentaires pour Elementor.
    • WooCommerce : Solution complète pour le commerce en ligne.
    • LiteSpeed Cache : Optimisation de la vitesse et performances du site.

Prérequis

  • Serveur local : XAMPP ou équivalent.
  • PHP version : 8.2.18.
  • MySQL (inclus dans XAMPP).
  • Navigateur web moderne.

Installation

Étapes d'installation :

  1. Clonez ce dépôt Git dans le dossier htdocs de votre installation XAMPP : 
    git clone https://github.com/KUNUZE/WordPressSite
  2. Placez le projet dans : 
    C:/xampp/www/wordpress
  3. Configurez la base de données :
    • Ouvrez phpMyAdmin via XAMPP.
    • Créez une base de données nommée wordpress-site.
    • Importez votre fichier .sql de sauvegarde (si disponible) dans cette base.
  4. Configurez les informations de connexion dans wp-config.php : 
    define('DB_NAME', 'wordpress-site');
define('DB_USER', 'root');
define('DB_PASSWORD', '');
define('DB_HOST', 'localhost');
  5. Démarrez Apache et MySQL depuis le panneau de contrôle XAMPP.
  6. Accédez à votre site en local : 
    http://localhost/wordpress

Structure du Projet

  • /wp-content/themes/blocksy : Thème utilisé pour personnaliser le site.
  • /wp-content/plugins/ : Liste des plugins installés :
    • Elementor
    • Premium Addons for Elementor
    • LiteSpeed Cache
    • WooCommerce

Fonctionnement

  • Connectez-vous au tableau de bord WordPress pour gérer le contenu : 
    http://localhost/wordpress/wp-admin
  • Utilisez les identifiants administratifs configurés lors de l’installation.
  • Utilisez Elementor et les Premium Addons pour personnaliser vos pages.
  • WooCommerce est déjà installé pour gérer votre boutique en ligne :
    • Configuration des produits : Produits → Ajouter un produit.
    • Gestion des commandes : WooCommerce → Commandes.

Performances

Pour des performances optimales, LiteSpeed Cache a été configuré pour optimiser la vitesse de chargement des pages.
